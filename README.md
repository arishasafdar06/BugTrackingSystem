# Bug Tracking System

A Java console-based Bug Tracking System developed using Java and MySQL.

## Features
- User Registration
- User Login
- Report Bugs
- View All Bugs
- Update Bug Status
- Delete Bugs
- MySQL Database Integration (JDBC)

## Technologies Used
- Java
- MySQL
- JDBC
- VS Code

## Project Structure
```
src/
├── dao/
├── database/
├── model/
├── Main.java

lib/
└── mysql-connector-j-9.7.0.jar

database_script.sql
README.md
```

## Database Setup
1. Start MySQL using XAMPP.
2. Create a database named `bug_tracking`.
3. Import the `database_script.sql` file.
4. Run the project from `Main.java`.
